# AlgoTN — Apprendre l'algorithmique

Plateforme interactive (HTML/CSS/JS, 100% statique) pour le cours Algorithmique & Structures de Données — programme TI (ISET Sidi Bouzid).

## Contenu
- Chapitres clés: Récursivité, Pointeurs, Listes chaînées
- Visualiseurs animés: pile d'appels, mémoire/pointeurs, liste chaînée
- QCM type examen (mode libre + mode examen minuté), exercices corrigés, fiches de révision
- Thème clair/sombre, recherche (Ctrl+K), progression sauvegardée localement

Ouvre `index.html` ou visite la page GitHub Pages.

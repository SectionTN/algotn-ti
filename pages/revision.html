⏰ Dernière heure avant l'examen

Les 12 réflexes qui sauvent

①
Pointeur = variable qui contient une adresse. &n = adresse de n ; p^ = valeur pointée.
②
Après Liberer(p), p est pendant : utiliser p^ = ERREUR (piège récurrent).
③
Allouer sans Liberer = fuite mémoire. Nil = « pointe sur rien ».
④
Récursivité = cas de base + cas récursif (plus petit). Pas de base → débordement de pile.
⑤
Sur une liste : base = L = Nil ; appel récursif sur L^.suivant.
⑥
Nœud = valeur + suivant. Fin de liste = Nil. Tête = pointeur sur le 1ᵉʳ nœud.
⑦
Insertion en tête : p^.suivant ← L PUIS L ← p (jamais l'inverse).
⑧
Suppression : garde prec + gère le cas tête (prec = Nil) + Liberer.
⑨
Tableau = accès O(1) mais taille fixe ; Liste = O(n) mais dynamique.
⑩
Fonction renvoie une valeur (Retourner) ; Procédure agit (souvent via var).
⑪
Modifier l'argument de l'appelant ⟹ passage par adresse / var, pas par copie.
⑫
Complexité utile : factorielle O(n), Fibonacci naïf O(2ⁿ), parcours liste O(n).

∞ Récursivité

  • 2 parties : cas de base + cas récursif.
  • Descente (empile) → remontée (dépile/combine).
  • Hypothèse de récurrence : suppose l'appel sur (n−1) correct.
  • Pièges : pas de base, problème non réduit, oublier Retourner.
